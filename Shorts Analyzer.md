# Shorts Analyzer

## Overview

A Python script that reads an Excel file containing a column of URLs (Instagram Reels, YouTube Shorts, and TikTok).

## Features

- Reads URLs from an Excel file
- Scrapes or fetches engagement metrics for each link:
  - Views
  - Comments
  - Likes
  - Saves
  - Shares
- Writes the fetched values into adjacent columns to the right of each link
- Handles different URL formats for each platform (Instagram Reels, YouTube Shorts, TikTok)
- Includes basic error handling for broken or private links
